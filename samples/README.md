# Samples (synthetic)

These files are **fabricated demonstrations only** — they contain **no real or client data**.
They exist to illustrate the input/output format and the audit report.

| File | Shows |
|------|-------|
| `sample_input.txt` | One raw record in the standard (public) hand-history text format. |
| `sample_output.txt` | The same record after Stage 3 (identity isolation) + Stage 4 (perspective duplication) — illustrative. |
| `audit_report_sample.csv` | The shape of a per-file validation report: `original` vs `expected` vs `actual` counts, with `status = OK` / `EMPTY_OK` and zero mismatches. |
